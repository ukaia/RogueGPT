# RogueGPT Launchers

This directory contains shortcut files to easily launch RogueGPT in CLI mode.

## Usage

### macOS
Double-click on `RogueGPT-CLI-mac.command` to launch the CLI version.

### Windows
Double-click on `RogueGPT-CLI-Windows.bat` to launch the CLI version.

### Linux
Run `./launch-cli.sh` from the terminal.

## Prerequisites
Make sure you have Node.js installed and have run `npm install` in the project directory first.

## Troubleshooting
If the launcher doesn't work:
1. Open Terminal (macOS) or Command Prompt (Windows)
2. Navigate to this directory
3. Run `npm run dev -w packages/cli` directly
